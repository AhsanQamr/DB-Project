
ABOUT PROJECT
---------------------------------------

Degree Clearance website is made using ASP.NET , C# and sql for the Database Layer.
This project was made on IDE Visual Studio 2022 and uses SSMS (sql server management studio)
for the Sql and Database Layer.

-----------------------------------------


INSTALLATION GUIDE
-----------------------------------------

Open the OneStop.sln file in Visual Studio 2019 or above
Connect Project.sql file with Visual Studio by going to View -> Connect DataBase.
Update your Connection String with each query written in the project
Finally go to Web.Config file and update Connection string tag
<Connection String>  'Update Connection String' </Connection String>
